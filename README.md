Bright AI Assistant คือโปรเจกต์เว็บแชทบอท AI ที่ออกแบบให้มีหน้าตาคล้ายระบบแชทสมัยใหม่ เช่น ChatGPT โดยเน้นการใช้งานจริงทั้งบนคอมพิวเตอร์และมือถือ

โปรเจกต์นี้ถูกสร้างขึ้นเพื่อแสดงความสามารถด้านการออกแบบหน้าเว็บ การจัดการ UI/UX ระบบล็อกอิน ประวัติห้องแชท การปรับแต่งโปรไฟล์ และการเชื่อมต่อกับ AI backend ผ่าน Flask API และ Ollama

Repository นี้ใช้สำหรับแสดงส่วน Frontend ผ่าน GitHub Pages ส่วน Backend ของ AI ต้องรันแยกบนเครื่อง local หรือ server

# English

## Project Overview

Bright AI Assistant is a web-based AI chatbot interface designed with a modern ChatGPT-style user experience.
The project focuses on responsive UI design, authentication flow, chat history, profile customization, theme switching, and AI backend integration.

This repository mainly contains the frontend files hosted on GitHub Pages.
The AI backend must be run separately using Flask and Ollama.

---

## Key Features

* Responsive chatbot web interface
* Desktop and mobile support
* Login and register UI
* User profile display
* Profile image upload
* Dark / Light theme switching
* New chat creation
* Recent chat history
* Rename chat rooms
* Delete chat rooms
* Image, file, and microphone buttons
* Designed for Flask API connection
* Designed to support user-based memory

---

## Tech Stack

* HTML
* CSS
* JavaScript
* Supabase Auth
* Flask API
* Ollama Local AI Model
* Local JSON Memory System
* GitHub Pages

---

## Purpose

This project was created as part of my portfolio to demonstrate skills in web development, AI integration, API connection, authentication UI, chatbot interface design, and user-based memory system planning.

The goal is to show a complete workflow from frontend design to backend AI communication.
